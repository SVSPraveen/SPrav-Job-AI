# Frontend React Application

This directory contains the user interface for the SPrav Job Application pipeline. 

It is a React application built with Vite that provides a command center dashboard, allowing you to:
- Monitor telemetry (jobs discovered, auto-applied, interviews, rejections).
- View the Master Job Portal and Application Scope configuration.
- Perform manual overrides in the "Action Required" and "Human Apply" queues.
- Configure system thresholds (ATS threshold, Fit score threshold, target salary).
- Rebuild the knowledge base from uploaded sources via the Onboarding flow.
